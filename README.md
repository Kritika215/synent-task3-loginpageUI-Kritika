# 🚀 Modern Glassmorphism Login Page UI

A modern, responsive, and visually appealing **Glassmorphism Login Page UI** built using **HTML5, CSS3, and JavaScript**. This project features a premium user interface with smooth animations, glassmorphism effects, responsive design, and interactive form validation.

It is ideal for **college projects, internships, portfolios, and frontend development practice**.

---

## ✨ Features

- 🔥 Modern Glassmorphism Design
- 🌄 Full-Screen Background Image
- 🎨 Gradient Overlay
- 📱 Fully Responsive Layout
- 🌙 Dark Elegant Theme
- 👁️ Show / Hide Password
- 💪 Password Strength Indicator
- 📧 Email Validation
- 🔐 Password Validation
- 💾 Remember Me (Local Storage)
- ⚡ Loading Animation
- 🎉 Login Success Animation
- ❌ Error Shake Animation
- 🌊 Smooth Hover Effects
- 💎 Ripple Button Effect
- 🎯 Smooth Page Transitions
- 🚀 Floating Login Card Animation
- 🎭 Modern UI/UX Design
- 🖥️ Cross-Browser Compatible
